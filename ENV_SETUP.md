# Environment Configuration

## Backend Environment (.env)

Create a `.env` file in the backend folder (optional):

```

RECIPIENT_EMAIL=shashankhsg@gmail.com
FRONTEND_URL=http://localhost:3000
```

## Frontend Environment (.env)

Create a `.env` file in the frontend folder (optional):

```
VITE_API_URL=http://localhost:8000
```

Note: The app is pre-configured to work with default localhost ports.
